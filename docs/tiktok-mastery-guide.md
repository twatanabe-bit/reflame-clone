## 1. TikTokというメディアの本質

### 1.1 他SNSとの決定的な違い
- **フォロワー数より動画単位の戦い**: 1本がフォロワー0でも100万回再生し得る。
- **「興味グラフ」ベース**: フォローではなく「視聴行動」でレコメンドが決まる（Instagramは社交グラフ寄り）。
- **完全没入型 UI**: スワイプとループ再生を前提に設計されている。
- **音源駆動**: 動画ではなく「音」がトレンドを作る世界唯一のプラットフォーム。

### 1.2 プロが捉える3つのレイヤー
1. **エンタメ層** — 暇つぶし、笑い、感動（最大母数）
2. **学び層** — 役立つ・タメになる（保存される）
3. **共感層** — 自分の物語・価値観（フォロー・コメントされる）

→ プロは1本の動画に **「即効性のエンタメ + 学び or 共感」** を必ず組み込む。

## 2. アルゴリズムの仕組み（2026年最新版）

### 2.1 評価される5大シグナル（重要度順）
1. **視聴完了率（Completion Rate）** — 最強指標。短尺ほど有利。
2. **平均視聴時間 / ループ率** — 2周以上見られると爆伸びトリガー。
3. **コメント率** — 「なぜ？」「どっち派？」型で誘発。
4. **シェア率** — 「友達に見せたい度」。シェアは最高評価。
5. **保存率** — 学び系・How-to系で重要。

> いいねは「ボーナス」程度。完了率・コメント・シェアが3本柱。

### 2.2 配信フェーズ
```
[投稿]
  ↓ Phase 1: 200〜500人テスト配信
  ↓ Phase 2: 1,000〜5,000人へ拡張（合格時）
  ↓ Phase 3: 10,000〜100,000へ
  ↓ Phase 4: バイラル（数十万〜数百万）
```
**各フェーズで完了率・エンゲ率が閾値を超えないと止まる**。初動2時間が勝負。

### 2.3 2026年の新傾向
- **長尺（60秒〜3分）**の優遇が顕著（広告収益の関係）。
- **AI生成コンテンツ判定**が厳格化 → 顔出し or ナレーション付きが優位。
- **検索流入**（TikTok SEO）が増加。キャプション・字幕にキーワード必須。
- **横型動画**も解禁され、YouTubeとのクロス運用が現実的に。

## 3. アカウント設計 — ニッチ・ペルソナ・コンセプト

### 3.1 ニッチ選定の黄金式
```
得意 × 需要 × 継続できる × マネタイズ動線
```
- 「広すぎる」は死。例: ×「料理」→ ◎「一人暮らし男性の3分タンパク質飯」
- 検索ボリュームをTikTok検索バーで確認（候補ワードが出るか）。

### 3.2 1センテンス・コンセプト
> 「**◯◯な人が、△△で、□□になる**アカウント」

例: 「**料理初心者の男**が、**3分・3工程・3材料**で、**コンビニ卒業**できるアカウント」

### 3.3 プロフィール3点セット
- アイコン: 顔 or シンボル1色背景（小さくても識別可能）
- 名前: 「肩書 / カテゴリ + ニックネーム」
- bio: 「誰に・何を・どうなる」を15字×3行で。リンクは1本に集約。

## 4. 「最初の2秒」フックの科学

### 4.1 視聴維持率は2秒で50%脱落する
脱落させない9つのフック型:

| # | 型 | 例 |
|---|---|---|
| 1 | 結論先出し | "結論、これやめれば痩せます" |
| 2 | 否定型 | "9割が間違ってる◯◯" |
| 3 | 数字 | "3秒で分かる" |
| 4 | 損失回避 | "知らないと損する" |
| 5 | 視覚的異常 | 想定外の動き・色・配置 |
| 6 | 問いかけ | "これ何だと思う？" |
| 7 | 暴露 | "誰も言わないけど…" |
| 8 | ストーリー導入 | "実は私、昔…" |
| 9 | 逆張り | "みんな◯◯って言うけど違う" |

### 4.2 2秒フックのチェック項目
- [ ] 最初のフレームに「動き」or「テキスト」or「顔の表情」がある
- [ ] 1秒以内に音が鳴る
- [ ] 「続きが気になる」設計になっている
- [ ] ターゲットが自分事化できるワードを含む

## 5. 動画構成テンプレート9選

### 5.1 黄金構成「PFREC」
**P**roblem → **F**ick(フック) → **R**eason → **E**xample → **C**all to action

### 5.2 即効テンプレート

1. **3 Tips型** — 「◯◯の方法3つ」(保存数◎)
2. **Before/After型** — 変化を見せる（完了率◎）
3. **疑似ストーリー型** — 「ある日、◯◯が起きた」(コメント◎)
4. **List Up型** — 「絶対NGな◯◯ 5選」(シェア◎)
5. **比較型** — 「Aは◯◯、Bは△△」(議論◎)
6. **暴露・裏側型** — 「業界の真実」(シェア◎)
7. **挑戦・実験型** — 「30日◯◯してみた」(フォロー◎)
8. **共感あるある型** — 「◯◯な人にしかわからない」(コメント◎)
9. **チュートリアル型** — 「保存版・◯◯のやり方」(保存◎)

### 5.3 ループ設計（必須テク）
動画の **冒頭 = 末尾** にして「気付かないうちに2周目」を作る。
完了率200%超え動画はほぼこの設計。

## 6. 撮影・編集ワークフロー

### 6.1 機材ミニマム構成
- スマホ（iPhone 12以降 or Pixel 7以降推奨）
- リングライト or 自然光（窓際）
- ピンマイク（風切り音と環境音対策、3,000円〜で十分）
- 三脚 or スマホスタンド

### 6.2 撮影の鉄則
- **縦9:16・1080×1920**で撮る
- **明るさは「肌が白飛びしない最大値」**
- **音声は別撮り or ピンマイク必須**（内蔵マイクはNG）
- 1テイクで撮らず、**B-rollを最低3カット**用意

### 6.3 編集の原則
- カット間隔 **平均1.5〜2.5秒**（飽きさせない）
- テロップは **画面の上1/3 or 下1/3** に配置（UIに被らない）
- 字幕は**喋りと完全シンク**（自動字幕→修正）
- 効果音は**フック・オチ・転換**の3か所だけ

### 6.4 推奨アプリ
- **CapCut**（テンプレ・自動字幕・トレンドエフェクト）
- **VLLO**（細かい調整）
- **TikTok内エディタ**（音源との親和性◎）

## 7. 音源・トレンド戦略

### 7.1 音源は「アルゴリズムの追い風」
- 急上昇音源を使うと配信範囲が広がる傾向。
- **「↑矢印マーク」がついた音源**を毎日チェック。
- 使うのは「**急上昇中だが飽和前**」(再生数1万〜10万件の音源)。

### 7.2 トレンド察知の3つの場所
1. **For Youを毎日30分スクロール**（業界ニッチで絞ったアカウントで）
2. **Creative Center → トレンド**（公式の急上昇データ）
3. **海外（特に米国）For Youを2週間先取り** ※VPN利用

### 7.3 リミックス文化を使う
オリジナル投稿に対し「Stitch / Duet / リミックス」で乗っかると、元動画の流入を受けられる。

## 8. 投稿運用とKPI管理

### 8.1 投稿頻度
- **最低: 1日1本 × 90日**
- 立ち上げ期は **1日2〜3本**（朝・昼・夜の中から2枠）

### 8.2 投稿時間帯（日本市場）
- 平日: 7:30 / 12:15 / 19:00 / 21:30
- 週末: 10:00 / 14:00 / 21:00

※自分のアナリティクス「フォロワーアクティブ時間」で最終調整。

### 8.3 計測すべきKPI

| 指標 | 合格ライン | 改善方法 |
|---|---|---|
| 平均視聴時間 | 動画尺の80%↑ | フック・テンポ・ループ |
| フル視聴率 | 30%↑ | 短尺化 / ループ |
| シェア率 | 1%↑ | 共感・有益・驚き |
| プロフィールアクセス率 | 2%↑ | キャラ立ち / 続きの示唆 |
| フォロー転換率 | 5%↑（プロフ到達者中） | bio / 過去動画の質 |

### 8.4 改善ループ（PDCAではなくOODA）
**観察 → 仮説 → 実行 → 検証** を**動画単位で**。週次レビューはNG、即日改善。

## 9. グロースハック実例集

- **シリーズ化**: 「Part 1, 2, 3…」でフォロー誘導。
- **コメント返し動画**: コメントを質問化して2本目以降のネタに。
- **固定動画3本**戦略: プロフ訪問者の離脱を防ぐベスト3を固定。
- **Stitchベイト**: 「あなたの◯◯を教えて」で UGC を呼ぶ。
- **コラボ**: 同規模クリエイターと相互Duet。
- **検索最適化**: キャプションに検索ワード3つ + 字幕にも入れる。
- **ハッシュタグ**: 大2 + 中3 + ニッチ2 の **7個構成**。

## 11. シャドウバン・違反対策

### 11.1 よくあるNG
- 他SNSロゴ・URL露出（特にYouTube）
- 過度な煽り・誇大表現
- 著作権音源（商用利用時）
- ダイエット薬・医療・投資の断定表現
- 同一動画の連続再投稿

### 11.2 シャドウバン疑い時のチェック
- ハッシュタグ検索で自分の動画が出るか
- 他人のブラウザで動画URLが見られるか
- 再生数が突然「平均の1/10以下」になっていないか

→ 該当時は **3〜7日投稿停止 + 過去動画の問題箇所削除**で回復することが多い。

## 13. ケーススタディ

### Case A: 料理系（0→10万フォロワー / 90日）
- ニッチ: 「一人暮らし・自炊未経験・男性」
- 構成: 3工程ループ + 完成サムネを冒頭に
- 勝因: **冒頭1秒で「完成形」を出して結末予告**

### Case B: 学び系（0→30万 / 120日）
- ニッチ: 「英語・聞き取れない発音だけ」
- 構成: 「これ何て言ってる？→答え→練習」
- 勝因: **クイズ形式でコメント率5%超**

### Case C: ストーリー系（0→100万 / 6ヶ月）
- ニッチ: 「副業失敗→再起の日記」
- 構成: シリーズ連載・日付入りタイトル
- 勝因: **「続き気になる」を毎本仕込み、フォロー必須化**

## 14. よくある失敗と対処

| 失敗 | 原因 | 対処 |
|---|---|---|
| 再生が伸びない | フックが弱い / ニッチが広い | 冒頭2秒の作り直し |
| フォローされない | キャラ立ちが弱い | bio・固定動画の刷新 |
| バズった後に伸び止まる | 期待値ギャップ | バズ動画の文脈で次を作る |
| コメントが荒れる | 煽り構成 | 共感型に転換 |
| 続かない | 完璧主義 | 「30本捨てる」覚悟 |
