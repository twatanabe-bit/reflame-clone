# Part 7｜90日 / 180日 / 360日 アクションプラン

ブランド戦略は「実行に移す具体タスク」がなければ意味がない。本パートは、本フレームワーク全体を実装に翻訳する。

---

## 7.1 全体タイムライン（俯瞰図）

```
[Day 0]      → 経営承認・キックオフ
[Day 1-30]   → 基盤整備フェーズ（ガイドライン整備・社内浸透開始）
[Day 31-60]  → 主要施策稼働フェーズ（チャネル別の本格稼働）
[Day 61-90]  → 測定・改善フェーズ（初回ブランドリフト測定）
[Day 91-180] → 拡張フェーズ（Salesforce連携深化・新規メディア・採用ブランディング）
[Day 181-360] → 定着フェーズ（業界標準言説化・書籍企画・年次総括）
[Year 2+]    → 構造資産化フェーズ（一人当たり生産性業界トップ・グローバル事例化）
```

## 7.10 アクション完了の判定基準

各タスクは以下3要件を満たした時点で「完了」と判定：

1. **アウトプット**：物理的な成果物が存在する（URL・PDF・動画ファイル等）
2. **承認**：CMOまたは担当事業部長の承認スタンプがある
3. **計測開始**：成果物の効果計測が開始されている（KPIダッシュボードに反映）

## 7.11 アクションの遅延・未達時の対応【提案】

- 月次ブランドレビュー会議で未達タスクを必ず晒す（個人攻撃でなく、構造的原因を抽出）
- 3週遅延でCMOへエスカレーション
- 6週遅延で経営会議報告
- 90日プラン中の遅延率が30%を超えた場合は、戦略全体を再設計
