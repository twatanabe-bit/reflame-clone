## 5.2 AI活用ブランド資産化の戦略フレーム【提案】

### 5.2.1 「単発の登壇」を「構造資産」に変える4ステップ

```
[Step 1] イベント・登壇の発生
   ↓
[Step 2] コンテンツ化（記事・動画・スライド・データ）
   ↓
[Step 3] PR・SNS・採用・営業に再分配
   ↓
[Step 4] 業界の標準言説として定着
```

各登壇は「1回限り」のリソースを使うが、Step 2-4で **10倍-100倍のレバレッジ** が掛かる。

### 5.2.2 4ステップの実装例（Salesforce Agentforce World Tour Tokyo 2025-11-20）

| Step | 実装 |
|---|---|
| 1. 発生 | 登壇スピーチ実施（既存） |
| 2. コンテンツ化 | 登壇スライド/原稿を「Salesforce事例化レポート」としてPDF化、noteに完全版を寄稿、YouTubeに10分版アップ |
| 3. 再分配 | 中途・新卒・建築の全採用ピッチに「Agentforce World Tour Tokyo登壇企業」のスライド埋め込み／法人提案資料の冒頭に追加／TikTok短尺動画化 |
| 4. 定着 | 業界紙への寄稿依頼、HRTechイベントへの登壇連鎖、Salesforce社のさらなる事例認定取得 |

## 5.4 採用ブランディングでのAI資産活用【提案】

### 5.4.1 採用候補向けピッチでの権威性活用

採用ピッチの冒頭3分は **AI活用の権威性で固定**：

> 「REFLAMEはSalesforce社の事例として認定され、Agentforce World Tour Tokyoに登壇しました。35名規模の人材紹介企業がここまでAI活用で評価されるのは異例です。私たちの仕事は『AIに置き換わる仕事』ではなく『AIと一緒に再定義する仕事』です。」

### 5.4.2 メンバー入社後のAIリテラシー底上げ

- 入社1ヶ月以内に「AI活用研修」（既存：AI_CA_slides_v4ベース）
- 5大AI（ChatGPT / Claude / Perplexity / Gemini / DeepSeek）の使い分け教育
- Salesforce / Agentforce の実務トレーニング
- AI議事録・AIロープレなど社内ツールの使い方

これにより「入社後すぐAIネイティブで働ける」状態を作り、退職率低減＋採用ブランド向上の両立を狙う。

## 5.8 リスク管理【提案】

### 5.8.1 Salesforce依存リスク

REFLAMEのAI×権威性の文脈の多くがSalesforce由来。同社の戦略変化や事例化方針変更で打撃を受ける可能性。

**対策**：
- HubSpot / Anthropic（Claude API）/ OpenAI / Microsoft Copilot などの他プラットフォーム実装事例も並行して積み上げる
- 「Salesforce依存」ではなく「AI実装力で勝つ企業」にナラティブを移行する素地を作る

### 5.8.2 「AI使ってます」の陳腐化リスク

2-3年で「AI活用」自体は珍しくなくなる可能性。

**対策**：
- 「AI使ってる」から「AIで何ができている」への論点シフトを先取り
- 具体数字（マッチングスコア92点／求人ピックアップ時間ゼロ／面談前準備1/3）を常に提示
- 「人とAIの共存」という上位概念で次の文脈を作る

### 5.8.3 競合追随リスク

レバレジーズ・マイナビ等が同様の取り組みを始めた場合。

**対策**：
- 業界初・最速のポジションを取り続ける（年4回のアップデート発信）
- 「規模で負けても密度で勝つ」というカウンターストーリーを準備
- 競合の動きを月次でウォッチし、半歩先を行く施策を企画
