## このフレームワークの目的

「人とAIの共存」を牽引する人材業界のキーエンス的存在として、REFLAMEのコーポレートブランドと3事業部のブランドを統合的に再定義し、それを**マーケティング施策に翻訳して実装する**ための完全版ガイド。

具体的には以下4つの問いに、判断レベルではなく実装レベルで答える：

1. **REFLAMEは何者か** — コーポレートとして、世の中にどう認識されるべきか
2. **3事業部の役割と差別化** — 中途・新卒・建築は、それぞれどんなブランドで、どう接続するか
3. **何をやって、何をやらないか** — 各事業部のマーケ施策において「REFLAMEらしい」とは
4. **どう測るか** — ブランド指標を含むKPI体系
