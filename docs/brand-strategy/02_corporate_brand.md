# Part 1｜コーポレートブランド戦略

## 1.1 コアポジショニング

### 1.1.1 ポジショニングステートメント【提案・経営口頭起点】

候補者・企業・採用候補・投資家・業界関係者に対して、REFLAMEを30秒で説明する公式文：

> **株式会社REFLAMEは、AIと人の共存を最先端で実装する人材紹介企業です。Salesforce社が事例として認定し、Agentforce / SFUG CUP の場で登壇するほどの仕組み化を背景に、候補者一人ひとりに「キャリア選択」ではなく「人生の変化」を届けます。35名規模ながら一人当たり生産性で業界トップクラスを目指す、人材業界のキーエンス的存在です。**

### 1.1.2 構造（4要素）

| 要素 | 定義 | REFLAMEでの中身 |
|---|---|---|
| **For（誰のために）** | ターゲット | 20代を中心とした「変化を望む候補者」と「採用を変えたい成長企業」 |
| **Who（誰として）** | カテゴリ | 人材紹介企業 |
| **That（何で差別化）** | 差別化要因 | AIと人の共存による顧客体験 / 労働生産性の最大化 |
| **Unlike（誰と違って）** | 競合 | 「規模で勝つ大手」「価格で戦う格安」のいずれでもなく、「仕組みと密度」で勝つ第3軸 |

### 1.1.3 ポジショニングマップ【提案】

縦軸＝「規模 vs 密度」、横軸＝「テクノロジー活用度」で配置：

```
        ↑ 密度・伴走力（個に向き合う度合い）
        |
        |  REFLAME ★                     アサイン
        |     ↑                          ・hape
        |   差別化ゾーン
        |   「AIと人の共存」
        |
--------+---------------------------------→ テクノロジー活用度
        |
        |  マイナビエージェント            レバレジーズ
        |  DODA                          ・キャリアパーク
        |  リクルートエージェント         
        |                                
        ↓ 規模・大量送客
```

- 右下クラスター（大手・量重視）はAI活用に投資しているが、密度を捨てている
- 左上クラスター（ハイクラス特化）は密度は高いがAI活用に出遅れている
- **右上＝REFLAMEのホワイトスペース**：「人とAIの共存」で密度と仕組み化を同時に成立させる

### 1.1.4 ポジショニングの維持条件【提案】

このポジションは「言うだけ」では持続しない。維持するための3条件：

1. **AI活用の更新を年4回以上、外部発信する**（登壇・記事・事例）
2. **一人当たり生産性を毎期改善する**（FY26 → FY27 → FY28で逓増を可視化）
3. **「変化のストーリー」を月1本以上、候補者ベースで公開する**（声を匿名で集める仕組み）

## 1.2 PVV（Philosophy / Vision / Values）

### 1.2.1 Philosophy（フィロソフィー）【既存・不変】

> **全ての人に、人生を刺激的にするきっかけがある。**

- **解釈の核**: 「一部の優秀層への支援集中」ではなく「思いがある全ての人の底上げ」
- **この語の意味**: REFLAMEは「機会の不均衡」を否定する。学歴・経歴・業界・年齢の壁の前で、AIと人の力で「きっかけ」をフラットに届ける
- **対外発信での使い方**: 候補者向け1枚目スライド・採用ピッチ最初の30秒・登壇冒頭の必修フレーズ

### 1.2.2 Vision（ビジョン）【既存・不変】

> **"生存実感"を、日常に。**

- **解釈の核**: 「仕事＝苦行」を覆し「仕事＝生存実感」に書き換える
- **「生存実感」の言語化**【提案】: 自分の選択で人生が前に動いている感覚／他者と自分の差ではなく、過去の自分との差で測れる感覚／結果と過程の両方に納得できる感覚
- **対外発信での使い方**: コーポレートサイトのHeroコピー候補、YouTube/TikTokのチャンネル説明文

### 1.2.3 Values（バリュー）【既存】

主軸として確認されているもの：

- **Purpose over Numbers（ロマンと算盤）** — 短期効率と長期ロマンの両立
- **AIと人の共存による生産性最大化** — 単なる自動化ではなく、AIが人の成長機会を作る

【要確認】上記2つに加えて、5原則（後述）との関係性・統合方法を明文化する作業をPart 6で扱う。

## 1.5 ブランドの「人格」設計

### 1.5.1 ブランドペルソナ【提案】

REFLAMEを「もし1人の人だったら」で表現：

- **性別・年齢感**: 中性的（性別を強調しない）／30代前半相当
- **職業観**: 元事業会社の事業企画／元コンサル／元営業MVPの3つを混ぜたような実務家
- **話し方**: フォーマルすぎず、しかし軽すぎず。データと体験談を行き来する
- **得意なこと**: 相手の語っていることの裏側にある「本当の問い」を当てる
- **苦手なこと**: 表面的な共感／盛った言葉／曖昧な約束
- **口癖（OK）**: 「で、本当はどうしたいですか？」「数字でも、感覚でも、両方教えてください」「これは違いますね、ここがズレてます」
- **絶対言わないこと**: 「とりあえずやってみましょう」「気合いで頑張りましょう」「他社さんよりウチは安いです」

### 1.5.2 トーン&マナー（Voice）

| 場面 | トーン | 例 |
|---|---|---|
| **採用ピッチ・対候補者・対企業** | 誠実×知的×伴走的 | 「あなたの選択を、一緒に整理させてください」 |
| **登壇・PR・メディア取材** | 自信×実例×謙虚 | 「弊社の事例が、業界全体の参考になれば光栄です」 |
| **SNS（X/note）** | 知的×等身大×率直 | 「採用市場の○○、データで見ると実はこうなんです」 |
| **TikTok・YouTube** | 親近感×情報密度高 | 「これ、見落とすと年収100万違うやつです」 |
| **クライシス対応** | 誠実×事実ベース×即応 | 「事実関係を確認の上、本日中にお伝えします」 |

### 1.5.3 視覚的トーン&マナー【提案 / 要確認】

【要確認】カラーパレット・タイポグラフィ・ロゴ規定は別途VIガイドラインで詳細を策定。現時点での方向性：

- **カラー**: 既存のREFLAMEブランドカラーを継承（要確認）。アクセントとして「AIの未来感」を感じさせる青・紫グラデーションを限定使用
- **タイポグラフィ**: Noto Sans JP（本文）／英文はInter or Roboto。WeightはRegular/Bold/Blackの3階層に統一
- **写真**: スタジオ撮影よりドキュメンタリー寄り。CA・候補者の表情、会議室の様子、Salesforce画面のような「現場の温度」を優先
- **イラスト**: 抽象的なグラデーション・パーティクル表現は控えめに。図解・データ可視化を優先

## 1.6 ストーリーアーク（ブランドナラティブ）

REFLAMEの語るべき「物語の構造」を3幕構成で固定する。【提案】

### Act 1：旧世界の問題提起

- 日本は労働者人口不足という構造的危機にある
- 採用は「数の勝負」になり、候補者一人ひとりへの伴走が削られた
- AIで効率化された世界には「あなただけの選択」が失われた
- 求人紹介を行うだけならエージェントは要らない（【既存・登壇発言】）

### Act 2：REFLAMEの仮説と挑戦

- AIは人を置き換えるためではなく、人が「最も人にしかできない仕事」に集中するために使う
- そのために、Salesforceに候補者・企業の暗黙知を定量×定性で蓄積し直した
- Agentforceがハイパフォーマーの判断を再現するレベルにまで仕組み化した
- 結果、求人ピックアップは時間ゼロに、面談前準備は1/3以下に、密度は反比例で増えた

### Act 3：これから

- REFLAMEは「人とAIの共存」を一企業ではなく業界の標準にする
- 中途・新卒・建築という3つの事業領域で、それぞれの形で「変化のきっかけ」を量産する
- 35名規模の今、一人当たり生産性で業界トップクラスへ到達する
- 5年後、人材業界のキーエンスと呼ばれる存在になる

このストーリーアークは、登壇・採用ピッチ・コーポレートサイト・PR記事・社内総会で**全社員が共通言語**として使う。

## 1.8 競合認識（コーポレートレベル）

### 1.8.1 競合カテゴリの分類【提案】

REFLAMEの競合は「同じ事業内容の会社」ではなく「同じブランドポジションを取りうる会社」と定義する：

| カテゴリ | 代表企業 | REFLAMEとの違い |
|---|---|---|
| **大手規模戦略** | リクルートエージェント、DODA、マイナビエージェント | 規模で勝つ。密度では負ける |
| **若手大量送客** | レバレジーズ | スピードと量。AI活用度ではREFLAMEに分あり |
| **ハイクラス特化** | アサイン、hape Agent | 候補者の質に寄せている。AI活用と業界拡張では未来不透明 |
| **業界特化（建築）** | プレックス、CSC | 業界深耕は同等。AI×人の仕組み化では差別化可能 |
| **AI×HR新興** | （要監視）国内外のAIエージェント特化スタートアップ | テクノロジー直球。人材紹介ノウハウ蓄積では追いつかない |

### 1.8.2 競合に勝つための言語化【提案】

- 規模で勝てない相手には**「密度」で勝つ**（一人ひとりへの伴走深さ）
- 密度で同等の相手には**「AI仕組み化」で勝つ**（再現性とスケーラビリティ）
- 両方で同等の相手には**「権威性（Salesforce事例化）」で勝つ**（信頼の前借り）

→ 競合との対比で **常に2軸以上で優位** を示す。1軸では真似される。
