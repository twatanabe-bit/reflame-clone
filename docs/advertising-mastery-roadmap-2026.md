# 広告マスターロードマップ 2026年版
**未経験 → プロ広告運用者・クリエイティブストラテジストになるための完全カリキュラム**

> 対象: 広告未経験者 〜 中級者
> ゴール: 月予算1,000万円+の媒体運用、CMO/広告代理店のシニア職、独立フリーランス
> 推定期間: 集中学習で6〜12ヶ月、副業ペースで12〜24ヶ月

---

## 🧭 学習の3原則(2026年版)

1. **「媒体」より「人間」を学べ** — アルゴリズムは2年で陳腐化するが、消費者心理は普遍。
2. **AIをレバレッジに、ジャッジは人間に** — 2026年は「AIで作る」より「AIに何を作らせるか」が差別化要因。
3. **計測なき施策は宗教** — Incrementality(増分)で語れない人は卒業できない。

## 🧱 Phase 1: マーケティング基礎(3週間)

### 必修概念(暗記レベルで)
- **STP**(Segmentation, Targeting, Positioning)
- **4P / 4C / 7P**
- **AIDA / AISAS / AARRR / RACE**
- **カスタマージャーニーマップ**
- **LTV / CAC / Payback Period / Unit Economics**
- **NBD-Dirichlet モデル**(購買頻度分布)
- **メンタルアベイラビリティ × フィジカルアベイラビリティ**(Ehrenberg-Bass)

### 演習
1. 既存ブランド3つ(例: Apple, Uniqlo, Workman)のSTPを300字で記述
2. 自分の好きなSaaSのLTV/CACを推定計算
3. 友人/家族にインタビューしてカスタマージャーニーを描く

## ✍️ Phase 2: コピーライティング(4週間)

### コアスキル
- **大ベネフィット 1点突破**(機能ではなく便益)
- **PASTOR / PAS / AIDA / 4U** などのフレームワーク
- **訴求軸の網羅**(価格、時間、安心、欲望、所属、自己実現、恐怖、希少性)
- **ヘッドライン17パターン**(ケープルズ系)
- **マイクロコピー**(CTA、入力フォーム、エラー文)

### おすすめ教材
- Copyhackers(英語、Joanna Wiebe)— 世界標準
- Marketing Examples by Harry Dry(無料、Twitter/Web)
- 神田昌典『稼ぐ言葉の法則』
- 川上徹也『物を売るバカ』(ストーリーブランディング)

### 演習(毎週)
- **「広告写経」を毎日10分**: スワイプファイル(優良広告コレクション)を手書き
- 同じ商品で**訴求軸を5パターン**書き分ける(価格訴求/時短訴求/安心訴求/憧れ訴求/恐怖訴求)
- **見出しA/B案を最低7パターン**作る習慣

## 🎨 Phase 3: クリエイティブ制作(6週間)

### 2026年の必須スキル
- **縦型動画のフック設計**(0〜2秒で離脱を防ぐ)
- **UGC風広告**(プロが作るが素人に見える)
- **AI動画生成**: Sora 2, Runway Gen-4, Veo 3, Pika 2.0
- **AI画像生成**: Midjourney v7, Flux 1.1 Pro, Ideogram 3
- **AIアバター/UGC**: HeyGen, Arcads, Captions AI Studio
- **編集**: CapCut(必須)、Premiere Pro、After Effects
- **デザイン**: Figma、Canva Magic Studio

### 動画フレームワーク(2026版)
1. **3秒フック**(Pattern Interrupt / Problem / Curiosity Gap)
2. **共感**(ペルソナの本音を代弁)
3. **転換**(解決策の提示)
4. **証拠**(レビュー、データ、Before/After)
5. **CTA**(具体的・1つのみ)

### 演習
- 同一商品で**フックを20種**作って5秒動画化、最もCTRが高い型を発見
- AIで30本量産 → 人間が**勝ち筋を3本に絞る**プロセスを体験

### 2026年トレンド(必修)
- **AI完全自動化キャンペーン**(Advantage+ / PMax / Smart+)— 「設定しすぎない」運用
- **Signal Loss対応**: CAPI / Enhanced Conversions / SKAdNetwork 5 / Privacy Sandbox
- **First-party Data活用**: CDP、Customer Match、Lookalike代替手法
- **Creative Diversity**: 媒体が「学習データの幅」を要求するためクリエイティブ量産が運用の主戦場に

### 実践課題(必須)
- 自費で**5万円〜10万円**を1ヶ月運用
- 最低**3媒体**で同じ商品を運用 → CPA/ROASを比較レポート化
- 結果をNotion/Googleドキュメントで**ケーススタディ**化(ポートフォリオに使う)

### 身につけるべき思考

3. **ブランドガード**: AI出力のブランド毀損リスクを管理する仕組み

## 🔥 最後に — 2026年に効く「裏ロードマップ」

5. **ニッチで1番を取る**: 「D2C × 美容 × Meta」のような3軸交差で「日本一」を目指す
